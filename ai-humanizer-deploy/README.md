# AI Content Humanizer

This app detects AI-generated text and rewrites it to make it more human-like.

## Features
- AI Detection (AI score, perplexity, repetition)
- Intelligent rewriting using T5 model
- Decision-based processing pipeline

## Usage
1. Enter your text
2. View AI score and analysis
3. Get rewritten humanized output
